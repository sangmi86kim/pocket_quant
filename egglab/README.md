# 🥚 에그랩 (Egg Lab) — 새 포켓몬 부화 연구소

> 다음 알파는 여기서 나온다. 리그 v2가 증명했듯 **기존 6마리의 가중치 조정으로
> 짜낼 알파는 없다**(가중치 공간의 천장 = 현 챔피언). 천장을 올리려면 새 포켓몬.

## 컨셉

지수들을 **조합**해 새로운 시그널(포켓몬)을 부화시킨다. 단일 가격(QQQ)만 보던
기존 6마리와 달리, 알에서 나올 포켓몬은 **다른 지수가 가진 새 정보**를 본다.

## 알 후보 (부화 대기 — 지수 조합 재료)

| 알 | 재료 | 가설 (무슨 정보를 보나) |
|---|---|---|
| 공포알 | ^VIX (변동성 지수) | 옵션 시장이 매기는 공포 가격 — 실현변동성(VOL몬)보다 선행할 수 있음 |
| 금리알 | ^TNX (미 10년물 금리) | 2022형 금리발 하락은 가격보다 금리가 먼저 움직임 |
| 형제알 | QQQ/SPY 비율 | 성장주가 시장 대비 강한가 — 나스닥 자체의 상대 체력 |
| 대장알 | 반도체(SOX)/QQQ 비율 | 경기 민감 선봉대가 먼저 꺾이는가 |
| 달러알 | DXY (달러 인덱스) | 달러 강세 = 위험자산 역풍 국면 감지 |

## 부화 절차 (기존 심판 인프라 재사용)

```
① 알 정의      : 재료 지수 + 포지션 규칙(0~1 또는 이벤트형 기권)을 signals.py 문법으로
② 컨닝 검사    : tests/test_no_lookahead.py — 미래를 잘라도 과거 포지션 불변
③ 새 정보 검사 : tests/check_signals.py — 기존 6마리와 상관 낮아야 명단 가치 있음
                 (양의 상관 > 0.3이면 중복 포켓몬 = 부화 실패)
④ 미시 검증    : 발동 후 1/5/10/20일 경로 (2026-06-10 설계 기준 재사용)
⑤ 체육관 예선  : 6체육관 단판 — 기존 조합 대비 개선 있는가
⑥ 리그         : NSGA-III (웜스타트로 현 챔피언 선출전) → 챔피언로드 관문 ①·②
```

## 규칙

- **사천왕(hold-out)은 이미 개봉됨** — 새 포켓몬의 최종 판정은 지금부터 쌓이는
  미래 데이터로만 한다. 2020-07~ 구간은 '참고'까지만.
- 데이터는 `data.py` 캐시 체계 그대로 (yfinance 티커만 추가).
- 골든 넘버 심판이 기존 엔진 보호 — 새 포켓몬 추가가 기존 계산을 건드리면 즉시 적발.
