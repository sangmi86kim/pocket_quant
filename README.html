🎮 PocketQuant

GOTTA MONEY 'EM ALL — 모든 시장 국면을 제패하는 최강 전략 연구실

포켓몬 파티 구성을 포트폴리오 최적화 문제로 재해석한 CLI 프로젝트입니다. 전략 포켓몬(시그널 조합)을 만들고, 실제 시장 국면(체육관)에서 yfinance 실데이터로 백테스트해 HP/ATK/DEF/SKILL 4스탯을 뽑고, 유전 알고리즘(GA)으로 최적 조합을 탐색합니다.

📈 실데이터(SPY) 백테스트 기반이지만 투자 자문이 아닙니다 — 학습/연구용입니다.

🧬 유전자 (시그널)

각 유전자는 실제 지표 로직으로, 가격을 받아 그날의 포지션(0~1, 1=풀매수·0=현금)을 만듭니다. 전략 포지션 = 유전자들의 평균.

유전자성격로직 (포지션)
DD리스크드로다운 스탑 — 고점 대비 −10% 넘게 빠지면 현금화
RSI심리RSI(14) < 70이면 탑승, 과열이면 현금
MA추세가격 > 200일 이평이면 탑승
BB변동성볼린저 상단밴드 위(과열)면 현금
VOL시장 상태실현변동성 레짐 — 평온=탑승 / 중간=반반 / 격동=현금
MOM추세 강화모멘텀 — 최근 ~3개월 수익률이 양수면 탑승

예시: MA-VOL킹, DD-RSI몬, DD-MA-MOM몬, 타이탄 드래곤

💱 매매 방식 (어떻게 사고파나)

신호가 뜨면 "전부 사고/판다"가 아니라, 매일 "주식을 몇 % 들고 있을지(비중)"를 정해 거기에 맞춥니다.

  • 각 유전자가 매일 비중을 뱉습니다: 1.0=풀매수 · 0.0=전부 현금 · 0.5=반반
  • 전략 비중 = 가진 유전자들의 비중 평균
  • 매일 그 비중으로 리밸런싱 (비중이 그대로면 보유, 바뀌면 그만큼만 사고팜)
그날 수익 = (어제 비중) × (오늘 SPY 등락)

예) 어제 50% 들고 있었는데 오늘 SPY +2% → 내 자산 +1%. 현금 부분은 이자 0%. 신호는 종가로 확인하고 다음 날 반영합니다 (룩어헤드 방지 = 하루 lag).

⚠️ 이상적인 시뮬레이션입니다 — 수수료·세금·슬리피지 0, 매일 정확히 비중 조정, 소수점 주식 가능, 현금 이자 0% 가정. "어느 전략이 어느 시대에 강한가"를 비교하는 연구 도구이지 실거래 권유가 아닙니다. (자주 사고파는 전략은 수수료만 넣어도 수익이 확 깎임)

🏟️ 체육관 (시장 국면)

각 체육관은 실제 역사적 기간을 가리키며, 그 기간의 SPY 가격으로 백테스트합니다. (difficulty/volatility는 판정엔 안 쓰는 연출용 메타데이터.)

체육관기간성격
2008 금융위기2008-01 ~ 2009-06시스템 붕괴 대폭락 → 방어형 천국, 공격형 지옥
2020 코로나 급락2020-02 ~ 2020-06V자 급락 후 즉시 회복 → 버티면 생존
2017 불타입 상승장2017-01 ~ 2017-12저변동성 꾸준한 우상향 → 추세추종/모멘텀 강세
2015-2016 횡보장2015-01 ~ 2016-12방향 없이 출렁임 → 추세형 헛신호, 평균회귀 유리
💡 폭락(2008)·급락(2020)·상승(2017)·횡보(2015-16) 4국면을 일부러 섞었습니다. 한 전략이 네 국면을 다 잘하긴 어렵고, 이 충돌이 다목적 최적화가 필요한 이유입니다.

⚔️ 전투 로직 (백테스트 → 스탯)

1) 유전자들이 만든 일별 포지션(0~1)을 하루 lag 적용해 시장 수익에 곱함 (룩어헤드 방지)
2) 체육관 기간만 잘라 자산곡선에서 CAGR / 최대낙폭(MDD) / 샤프 / 평균현금 측정
3) 0~100 스탯으로 정규화:
     ❤️ HP    = 평균 현금 비중           (위기 버틸 체력)
     ⚔️ ATK   = CAGR                      (-25%→0, +25%→100)
     🛡️ DEF   = 1 − 내MDD/시장MDD         (시장보다 덜 빠진 정도)
     ✨ SKILL = 샤프비율                  (-1→0, 3→100)

종족치 BST = HP+ATK+DEF+SKILL (0~400). 등급은 적합도(스탯 가중평균, 0~100)로: S≥90 · A≥70 · B≥50 · C≥30 · D 그 외

🪙 "다 꼬라박아도 상폐만 아니면 진 게 아니다" — 생존/사망 이진판정을 버리고 전부 연속 스탯으로 잽니다. 페널티(0점 처리)는 없습니다.

📁 구조

3층 구조: main(입력) → service(흐름) → backend(계산)

pocket_quant/
├─ main.py              # CLI 입력만 받음 (argparse → service 호출)
├─ requirements.txt     # yfinance · pandas · numpy
└─ app/
   ├─ service.py        # 실행 흐름 조립 (단판/진화 순서 + 시드 + 출력)
   └─ backend/
      ├─ models.py      # dataclass (Stats/Strategy/Gym/Report) + 스탯 가중치/등급
      ├─ data.py        # yfinance 다운로드 + 디스크 캐시 + LoadedGym 로딩
      ├─ signals.py     # 유전자 = 진짜 지표 로직 → 일별 포지션(0~1)
      ├─ dex.py         # 포켓몬 도감 (유전자 설명 카드)
      ├─ strategy.py    # 전략 생성 + 이름 자동 생성
      ├─ gym.py         # 체육관 4종 (실제 시장 국면 기간)
      ├─ battle.py      # 실데이터 백테스트 → HP/ATK/DEF/SKILL 스탯 (순수 계산)
      └─ evolve.py      # 단일목적 GA (적합도=스탯 가중합)
